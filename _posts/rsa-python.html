Criptosistemas de clave pública: Ejemplo práctico con RSA

¿Qué hace este código?

Este ejemplo en Python muestra cómo usar RSA para cifrar y descifrar un mensaje.

Código Python:


from cryptography.hazmat.primitives.asymmetric import rsa, padding
from cryptography.hazmat.primitives import hashes

# Generar claves
private_key = rsa.generate_private_key(public_exponent=65537, key_size=2048)
public_key = private_key.public_key()

# Mensaje original
mensaje = b'Hola, este es un mensaje secreto'

# Cifrar el mensaje
mensaje_cifrado = public_key.encrypt(
    mensaje,
    padding.OAEP(mgf=padding.MGF1(algorithm=hashes.SHA256()), algorithm=hashes.SHA256(), label=None)
)

# Descifrar el mensaje
mensaje_descifrado = private_key.decrypt(
    mensaje_cifrado,
    padding.OAEP(mgf=padding.MGF1(algorithm=hashes.SHA256()), algorithm=hashes.SHA256(), label=None)
)

print("Mensaje original:", mensaje)
print("Mensaje descifrado:", mensaje_descifrado)

¿Cómo ejecutarlo?

  1. Instala la librería: pip install cryptography
  2. Guarda el código en un archivo rsa_demo.py
  3. Ejecuta el script: python rsa_demo.py
